# 💰 WealthWise - Personal Savings Tracker App

A modern and intelligent Personal Savings Tracker application that helps users manage their income, expenses, and savings goals efficiently. The application provides real-time financial insights, spending analysis, and savings forecasting through an intuitive dashboard and interactive visualizations.

## 🚀 Features

### 🔐 Authentication

* User Login & Registration
* Secure form validation
* Profile management

### 💵 Income & Expense Management

* Add, Edit, Delete, and View transactions
* Categorize expenses and income
* Transaction history tracking
* Search and filter transactions

### 🎯 Savings Goals

* Create multiple savings goals
* Track progress with visual progress bars
* Goal completion monitoring
* Goal-wise savings allocation

### 📊 Dashboard

* Total Balance Overview
* Total Income Summary
* Total Expenses Summary
* Total Savings Display
* Goal Progress Tracking
* Quick Financial Insights

### 📈 Reports & Analytics

* Monthly income and expense reports
* Spending pattern analysis
* Category-wise expense breakdown
* Interactive charts using Chart.js
* Savings growth visualization

### 🌙 User Experience

* Dark Mode & Light Mode
* Responsive Mobile-Friendly Design
* Smooth Animations
* Modern Card-Based Interface

### 🤖 Advanced Features

* AI-Powered Spending Insights
* Budget Recommendations
* Expense Prediction
* Savings Forecasting
* PDF Report Export
* Monthly Reminder Notifications
